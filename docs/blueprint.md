# **App Name**: TuneDrop

## Core Features:

- Music Search: Allow users to search for music tracks by title, artist, or keywords. Results fetched from a selected provider such as the iTunes API.
- Preview Playback: Enable users to preview music tracks before downloading to ensure it's the correct song or version.
- Download Management: Handle the download process of music files, including displaying progress and managing storage. Integrate a database like Cloud SQL to store the downloaded items.
- AI Playlist Creator: AI tool which will take song input from the user, along with a prompt describing the kind of mood or feeling of the desired playlist, then use AI to assemble and show the user a potential playlist which can then be assembled from the songs available via download.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2), evoking a sense of modernity, technology, and accessibility related to music.
- Background color: Light gray (#F0F2F5), providing a clean, neutral backdrop that makes the vibrant blue stand out.
- Accent color: Turquoise (#30D5C8) for interactive elements, offering a fresh and complementary touch to the primary blue.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text, offering a balance of modernity and readability.
- Use consistent, outline-style icons to represent actions and categories, maintaining a clean and modern appearance.
- Design a clean, card-based layout with rounded corners to showcase music tracks and details. Focus on a user-friendly experience that makes browsing easy.
- Incorporate subtle transitions and animations when music tracks load or are added to the download queue.
